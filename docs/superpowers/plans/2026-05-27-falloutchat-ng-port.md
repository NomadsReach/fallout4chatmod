# FalloutChat NG Port Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a self-contained `E:\F4SE OG\FalloutChat-NG\` project that is an exact copy of FalloutChat adapted to target Fallout 4 Next-Gen (v1.10.984+) using the Ryan-rsm-McKenzie CommonLibF4 fork as an embedded git submodule.

**Architecture:** New git repo at `FalloutChat-NG\`; all source files copied from `FalloutChat\`; CommonLibF4 added as a submodule inside the project (not a sibling); two small code changes (CMakeLists.txt path + runtime version constant); new build_dll.bat with NG deploy path.

**Tech Stack:** C++20, CMake, vcpkg (x64-windows-static-md), MSBuild VS 2022, CommonLibF4 NG (Ryan-rsm-McKenzie), ImGui DX11, MinHook, ixwebsocket
